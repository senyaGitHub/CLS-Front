This is a new React Native front for app made for hackathon.


```bash
# using npm
npm start

# OR using Yarn
yarn start
```

App was tested for android, APK worked fine without major bugs, not tested under IOS.

### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
